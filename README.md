## 🛠️ Setup & Installation

```bash
# Clone the repository
git clone https://github.com/FLung13/fredy-portfolio.git

# Navigate into the project
cd portfolio

# Install dependencies
npm install

# Run development server
npm run dev
```

## 📦 Build for Production

```bash
npm run build
```
